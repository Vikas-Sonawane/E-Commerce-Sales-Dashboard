# 📊 Bhagwati Store Sales Dashboard (2025)

An interactive, data-driven Excel dashboard analyzing the 2025 sales performance, customer demographics, and order fulfillment metrics for Bhagwati Store.

---

## ✨ Key Insights & Visualizations

* **Orders vs Sales:** Monthly trend comparison between order volumes and overall revenue.
* **Sales - Men vs Women:** Demographic breakdown highlighting a 64% female customer majority.
* **Order Status:** Fulfillment health showing a 92% successful delivery rate.
* **Top 5 States:** Regional performance tracking led by Maharashtra (2.99M) and Karnataka (2.65M).
* **Orders by Age & Gender:** Detailed generation matrix (Adult, Senior, Teenager) segmented by gender.
* **Sales Channels:** Marketplace distribution analysis across Amazon (35%), Myntra (23%), Flipkart (22%), and others.

---

## ⚙️ Interactive Features

* **Dynamic Slicers:** Real-time dashboard filtering by Product Category, Sales Channel, and Month.
* **Pivot Architecture:** Fully refreshable backend data structure linked across dedicated reporting tabs.

---

## 🛠️ Built With

* **Microsoft Excel:** Pivot Tables, Pivot Charts, and Slicer Filters.
